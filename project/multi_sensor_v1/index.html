Multisensor V1

Multisensor v1 - проект по созданию платы управления универсальными, а главное компактными датчиками для «Умного дома» на базе модулей ZigBee CC2530 от компании EBYTE E18-MS1-PCB.

На данный момент датчики можно использовать с ZigBee2MQTT, в будущем хочу добавить еще одну интересную систему ...

Для сборки датчика используется минимум деталей. Возможно подключение различных датчиков: датчик открытия двери и окна, датчик температуры, влажности, атмосферного давления, датчик движения, датчик уровня освещенности и различные другие цифровые и аналоговые датчики. Все это доступно при невысокой стоимости изготовления.

Ниже представлена принципиальная схема датчика.

Размер платы датчика получился 30.86мм х 26.16мм.

Используемые компоненты:

- Ebyte E18-MS1-PCB CC2530 1шт.

- Геркон (Опционально) 1шт.

- BME280 (Опционально) 1шт.

- PIR сенсор AM312 (Опционально) 1шт.

- Батарейный отсек CR2032 (Доступны 2 типа первый, второй) 1шт.

- Резисторы 0805 5шт.

- Конденсатор 0805 (Опционально) 1шт.

- LED 0805 1шт.

- Кнопка 3*4*2 1шт.
